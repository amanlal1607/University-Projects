To run the application you require:
1.JRE 6.0 or above
2.JVM
3.A keyboard :)

STEPS TO RUN:
1.Unzip the files in any directory of your choice
2.Shift+right click and Open command window in the current directory
3.Use the following commands to run the code : 

javac Minesweeper.java
java Minesweeper

-----The program is now running--------
enter your choice of rows
enter your choice of columns
enter the minefield to be computed
